## 1. 贡献流程和规范

参与开发前，请熟悉贡献流程及相关规范，详细请详见 [《贡献流程和规范说明》](../common/contribute_workflow.md)。

## 2. 贡献方法

本仓库为主仓，不包含具体工具代码。请根据您准备贡献的工具，前往对应工具仓库的 README 查阅详细的《贡献指南》：

| 工具名称            | 功能简介 |                                                源码仓库                                                 |

| **msProbe**       | 精度调试 |                             [点击查看](https://gitcode.com/Ascend/msprobe)                              |
| **TensorBoard**   | 分级可视 |           [点击查看](https://gitcode.com/Ascend/msprobe/tree/master/plugins/tb_graph_ascend)            |
| **msProf**        | 模型调优 |                              [点击查看](https://gitcode.com/Ascend/msprof)                              |
| **msprof-analyze**| 性能分析 |                          [点击查看](https://gitcode.com/Ascend/msprof-analyze)                          |

| **msPTI**         | 性能剖析 |                              [点击查看](https://gitcode.com/Ascend/mspti)                               |
| **msMemScope**    | 显存检测 |                            [点击查看](https://gitcode.com/Ascend/msmemscope)                            |
